Pinxel UIKit

Componentes base para personalización, comercio y producto digital.

Este UIKit traduce el brandbook de Pinxel en componentes HTML/CSS listos para documentar, prototipar y alimentar futuras interfaces. El CSS global contiene tokens y fundamentos; cada componente tiene su propia hoja de estilos.

Arquitectura

La estructura separa fundamentos de especificaciones para que cada componente pueda crecer sin contaminar el sistema completo.

CSS global

uikit/css/pinxel-uikit.css define tokens, layout documental, tipografía, grillas, tablas, stages y utilidades base.

CSS por componente

uikit/css/components/*.css contiene especificaciones independientes para botones, formularios, tarjetas, estados, navegación, datos y flujos.

HTML independiente

uikit/components/*.html documenta cada componente con anatomía, uso, estados y ejemplos de implementación.

Instrucciones para IA

ai/pinxel-ui-instructions.md define las reglas operativas que debe seguir cualquier agente antes de generar UI.

Registro de componentes

uikit/component-registry.md indica qué archivos consultar según el componente o patrón que se necesite crear.

Submodule versionado

Úsalo cuando el proyecto deba guardar una referencia exacta al commit del sistema de diseño y compartirla con el equipo.

Principios del UIKit

Pinxel debe sentirse claro y modular, pero también capaz de sostener productos físicos personalizados y automatizaciones con IA.

Light first Diseñar primero sobre superficies blancas o gris muy claro. El modo oscuro es opcional para herramientas largas y dashboards.
Pill actions Las acciones principales usan forma pill. Es una señal de marca: acción amable, táctil y clara.
Color funcional Púrpura para decisiones, naranja para impulso comercial, turquesa para progreso y estados positivos.
Personalización visible Los flujos de producto deben mostrar objeto, opciones, preview, datos del cliente y confirmación.
IA práctica La IA se presenta como asistente útil. Mostrar flujo, mensaje, estado o resultado antes que gráficos abstractos.
