Tamaños y color

El icono hereda el color del componente con currentColor. No usar rellenos decorativos ni combinar estilos.

Reglas

  • Usar Lucide como set base de iconos UI.
  • En React, usar lucide-react como componentes SVG; no icon font.
  • Usar SVG inline, sprite SVG o helper del theme/plugin en WordPress.
  • Importar SVG editable en Illustrator y Photoshop.
  • Usar currentColor para que el estado del componente controle el icono.
  • Los botones solo de icono necesitan aria-label.

No usar

  • No usar icon fonts como primera opción.
  • No usar icon font en React salvo integración legacy, requisito del cliente o dependencia externa que lo exija.
  • No mezclar Lucide con sets filled o estilos ilustrativos en la misma interfaz.
  • No usar iconos ambiguos para acciones críticas sin texto cercano.
  • No convertir Lucide en sistema de ilustración de marca.
  • No cambiar stroke, esquinas o proporción por icono de forma aislada.

Recursos

Usar estas fuentes oficiales para buscar, descargar, instalar o integrar iconos. En WordPress y sitios estáticos, priorizar SVG inline o sprite.

Buscar y descargar

Lucide Icons permite buscar iconos, copiar SVG y revisar nombres oficiales.

Guía oficial

Lucide Guide cubre instalación, accesibilidad, color, tamaño y stroke.

Paquetes

Lucide Packages lista paquetes oficiales para web, React, Vue, Svelte, Angular, Astro y static assets.

Repositorio

GitHub contiene el código fuente, releases, issues, contribuciones y licencia.

Figma

Lucide Icons plugin sirve para insertar iconos directamente en archivos de diseño.

Diseñar iconos

Icon Design Guide define canvas 24x24, stroke 2px, caps redondos, joins redondos y reglas de consistencia.

Código base

<button class="px-icon-button" type="button" aria-label="Buscar">
  <svg viewBox="0 0 24 24" fill="none" aria-hidden="true">
    <circle cx="11" cy="11" r="8" stroke="currentColor" stroke-width="2"/>
    <path d="m21 21-4.3-4.3" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"/>
  </svg>
</button>
