Esencia de marca

Pinxel debe sentirse como un ecosistema premium: claro, modular, humano y listo para convertir ideas en productos reales.

Clara

Reduce el ruido visual y verbal. Cada pantalla debe explicar qué está pasando y cuál es el siguiente paso.

Precisa

La interfaz no decora decisiones. Usa jerarquía, espaciado y estados para orientar con exactitud.

Optimista

El color aparece como energía medida: púrpura para dirección, naranja para impulso y turquesa para avance.

Logo

La firma principal vive limpia y sin efectos. El isotipo funciona como módulo de app, favicon y marca compacta.

Color

El sistema es light-first. Los acentos son potentes, pero se usan en dosis pequeñas para mantener elegancia y foco.
