# Pinxel Design System

Brandbook, AI instructions, and UIKit foundation for Pinxel products, websites, dashboards, and commerce workflows.

## Structure

- `about/`: Pinxel context and business definition.
- `ai/`: operational instructions for AI-generated UI.
- `assets/`: logo, symbol, and favicon.
- `brandbook/`: source brandbook in Markdown and visual HTML viewer.
- `uikit/`: component documentation, registry, global CSS, and component CSS.
- `references/`: external brand references used during exploration.

## AI Entry Points

Before designing or coding Pinxel interfaces, read:

1. `brandbook/pinxel-brandbook.md`
2. `about/about-pinxel.md`
3. `ai/pinxel-ui-instructions.md`
4. `uikit/component-registry.md`
5. `uikit/index.html`
6. Relevant files inside `uikit/components/` and `uikit/css/components/`

## Local Viewing

Open these files directly in a browser:

- `brandbook/pinxel-brandbook.html`
- `uikit/index.html`
